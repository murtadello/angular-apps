How to setup Angular JS Project?

First you should install angular CLI (Command Line Interface)

a) npm install -g @angular/cli

Second you should enter to your folder where you want to store your project

b)cd Desktop

Third create your project using this command
ng new my-first-app --no--strict

once you created your angular project, you should enter to your 
folder and run the command

ng serve 

How to create Component using CLI
>>ng g c NameofComponent
